Action expressions

Function actionQuery
Type: ACTION

DataGraph actionQuery(Expression)

Action query embedded in a server-side expression

Function alert
Type: CLIENT_SIDE_ACTION

Void alert(PlainText)

Client-side bootstrap alert

Function catch
Type: BUILTIN

Void catch(Object)

Allows to declare a sub action sequence to execute as rejection handler

Function delete
Type: ACTION

Void delete(DomainType...)

Deletes the given domain objects

Function error
Type: ACTION

Void error(PlainText)

An action that executes the given transition

Function fork
Type: BUILTIN

List fork(Object...)

Special action function that takes multiple sequences as input, runs them in parallel and waits for all of them to end.

Function merge
Type: ACTION

DomainType merge(DomainType)

Merges the give domain object with its persistent state, i.e. it fills in all the missing null values

Function navigateTo
Type: CLIENT_SIDE_ACTION

Void navigateTo(PlainText, Map)

AJAX Navigation action within an exceed application. To be used in views without process.

Function query
Type: ACTION

DataGraph query(Expression)

Query embedded in a server-side expression

Function store
Type: ACTION

Void store(DomainType...)

Stores the given domain object

Function syslog
Type: ACTION

Void syslog(Object...)

System log function. Server logs per slf4j, client to the console API

Function transition
Type: CLIENT_SIDE_ACTION

Void transition(PlainText)

An action that executes the given transition

Function when
Type: BUILTIN

Void when(Object)

A conditional branch in the action promise chain

Component Expressions

Identifier context

Type: Any

The current cursor context or transition context.

Function lazyDependency
Type: BUILTIN

Object lazyDependency(Object)

Marks first the expression argument as being lazily evaluated in terms of context dependency updates.The user can change the context variables within without it triggering view updates.

Identifier model

Type: xcd.view.ComponentModel

A reference to the current model

Identifier props

Type: Any

In a component expression, the props/attributes of the component (e.g. props.name)

Identifier vars

Type: Any

In a component expression, the current vars of the component

Boolean isNew(DomainType)

Function fieldId

Object popCursor(Object, Integer)

Shortens the path of a cursor by n levels, returning the cursor location to a parent or grandparent of the current location.
